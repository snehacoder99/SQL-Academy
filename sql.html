SQL Basics to Intermediate

Introduction to SQL

SQL (Structured Query Language) is a standard language for accessing and manipulating databases.

Basic SQL Commands

Here are some basic SQL commands:

SELECT

The SELECT statement is used to select data from a database.

SELECT column1, column2 FROM table_name;

INSERT

The INSERT INTO statement is used to insert new records in a table.

INSERT INTO table_name (column1, column2) VALUES (value1, value2);

UPDATE

The UPDATE statement is used to modify the existing records in a table.

UPDATE table_name SET column1 = value1, column2 = value2 WHERE condition;

DELETE

The DELETE statement is used to delete existing records in a table.

DELETE FROM table_name WHERE condition;

Intermediate SQL Concepts

JOIN

The JOIN clause is used to combine rows from two or more tables, based on a related column between them.

SELECT column1, column2 FROM table1 INNER JOIN table2 ON table1.common_column = table2.common_column;

GROUP BY

The GROUP BY statement is used to group rows that have the same values into summary rows, like "find the number of customers in each country".

SELECT COUNT(CustomerID), Country FROM Customers GROUP BY Country;

HAVING

The HAVING clause is used to filter records that work on summarized GROUP BY results.

SELECT COUNT(CustomerID), Country FROM Customers GROUP BY Country HAVING COUNT(CustomerID) > 5;

SQL Libraries and Tools

MySQL

MySQL is an open-source relational database management system.

-- Connect to a MySQL database
mysql -u username -p database_name;

PostgreSQL

PostgreSQL is a powerful, open-source object-relational database system.

-- Connect to a PostgreSQL database
psql -U username -d database_name;

SQLite

SQLite is a C-language library that implements a small, fast, self-contained, high-reliability, full-featured, SQL database engine.

-- Open an SQLite database
sqlite3 database_name.db;
